## 账户与登录

<AccordionGroup>
  <Accordion title="忘记密码怎么办？">
    如果您忘记了登录密码，请按以下步骤重置：

    1. 前往 Google Media 登录页面，点击**忘记密码**链接。
    2. 输入您注册时使用的电子邮箱地址。
    3. 系统将向该邮箱发送一封密码重置邮件，请在 30 分钟内点击邮件中的重置链接。
    4. 按照页面提示设置新密码。

    <Tip>如果您未收到重置邮件，请检查垃圾邮件文件夹，或确认输入的邮箱地址是否正确。</Tip>
  </Accordion>

  <Accordion title="可以用同一账号在多台设备登录吗？">
    可以，Google Media 支持多设备同时登录，具体限制如下：

    - **免费版（Free）**：最多支持 **2 台**设备同时在线。
    - **高级版（Premium）**：最多支持 **5 台**设备同时在线。

    如果您尝试在超出限制的设备上登录，系统将提示您退出已登录的某台设备后再继续。
  </Accordion>

  <Accordion title="如何更改注册邮箱？">
    您可以通过以下步骤更改账户绑定的注册邮箱：

    1. 登录后进入**账户设置**页面。
    2. 点击**安全设置** > **更改邮箱**。
    3. 输入您的新邮箱地址并提交。
    4. 系统将分别向**旧邮箱**和**新邮箱**各发送一封验证邮件，您需要完成两次验证后更改才会生效。

    <Warning>请确保新邮箱地址有效且可正常接收邮件，否则验证链接将无法送达，更改操作将无法完成。</Warning>
  </Accordion>
</AccordionGroup>

## 订阅与付款

<AccordionGroup>
  <Accordion title="免费版和高级版有什么区别？">
    免费版与高级版在功能和内容访问权限上存在明显差异：

    | 功能 | 免费版 | 高级版 |
    |------|--------|--------|
    | 每日新闻阅读 | 有限制 | 无限制 |
    | 深度报道与独家内容 | 不可访问 | 完整访问 |
    | 历史档案查阅 | 近 90 天 | 全部档案 |
    | 多设备同时登录 | 2 台 | 5 台 |
    | 广告展示 | 有 | 无 |
    | 客服响应优先级 | 标准（1-2 个工作日） | 优先（4 小时内） |

    如需了解完整的功能对比和订阅价格，请访问[订阅方案页面](/subscriptions/plans)。
  </Accordion>

  <Accordion title="支持哪些支付方式？">
    Google Media 目前支持以下支付方式，方便国内外用户灵活选择：

    - **微信支付**
    - **支付宝**
    - **Visa** 信用卡 / 借记卡
    - **Mastercard** 信用卡 / 借记卡
    - **PayPal**

    <Info>所有支付交易均通过加密通道处理，您的支付信息将受到严格保护。</Info>
  </Accordion>

  <Accordion title="如何取消自动续费？">
    如果您希望关闭订阅的自动续费功能，请按以下步骤操作：

    1. 登录账户，进入**账户设置**。
    2. 点击**订阅管理**。
    3. 找到当前订阅计划，点击**关闭自动续费**。
    4. 确认操作后，系统将发送一封确认邮件至您的注册邮箱。

    关闭自动续费后，您的高级版权益将持续到当前订阅周期结束，届时账户将自动降级为免费版。
  </Accordion>

  <Accordion title="退款政策是什么？">
    我们提供以下退款保障：

    - 自购买之日起 **7 天内**，如您尚未访问任何高级付费内容，可申请全额退款。
    - 如已访问高级内容，将无法申请退款。
    - 退款申请请通过[联系客服](/support/contact)页面提交，并注明订单编号。

    <Note>自动续费产生的订单同样适用上述退款政策。建议您在续费日期前提前关闭自动续费，以避免不必要的扣款。</Note>
  </Accordion>
</AccordionGroup>

## 内容访问

<AccordionGroup>
  <Accordion title="为什么有些文章需要付费？">
    Google Media 致力于为读者提供高质量的新闻内容。部分深度报道、独家专访及调查性新闻需要大量的采编资源投入，因此仅对**高级版（Premium）**订阅用户开放。

    付费内容通常以锁形图标或"高级"标签标注，免费版用户可阅读文章导言部分，如需阅读全文，请升级至高级版订阅。
  </Accordion>

  <Accordion title="如何访问历史档案？">
    根据您的订阅等级，历史档案的访问权限如下：

    - **免费版**：可查阅近 **90 天**内发布的文章及档案内容。
    - **高级版**：可访问自平台上线以来的**全部历史档案**，不受时间限制。

    如需查阅特定日期的历史内容，您可以使用搜索功能并通过日期筛选器进行精确查找。
  </Accordion>

  <Accordion title="内容支持哪些语言？">
    Google Media 作为连接中国与世界的国际媒体平台，主要提供以下语言的内容：

    - **简体中文**：平台主要内容语言，覆盖全部频道。
    - **繁体中文**：大部分文章支持繁体中文版本，可在阅读页面切换。
    - **英文**：部分精选文章提供英文版本，以服务国际读者。
    - **双语对照**：部分重点报道提供中英双语对照版本，方便语言学习者阅读。

    您可以在账户设置中将默认语言设置为您的偏好语言。
  </Accordion>
</AccordionGroup>

## 技术问题

<AccordionGroup>
  <Accordion title="应用程序崩溃或加载缓慢怎么办？">
    如果您遇到应用崩溃或加载缓慢的问题，请依次尝试以下解决方法：

    1. **检查网络连接**：确保您的设备处于稳定的 Wi-Fi 或移动数据网络下。
    2. **清除缓存**：进入应用设置，清除本地缓存文件，释放存储空间。
    3. **更新应用版本**：前往 App Store 或 Google Play，检查是否有可用的更新，安装最新版本。
    4. **重启设备**：完全关闭并重启您的手机或平板，再次打开应用。

    如果问题仍然存在，请访问[故障排除页面](/support/troubleshooting)获取更详细的解决方案。
  </Accordion>

  <Accordion title="收不到推送通知怎么办？">
    如果您未能收到 Google Media 的推送通知，请按以下步骤检查：

    1. **检查系统通知权限**：
       - **iOS**：进入"设置" > "通知" > "Google Media"，确认已开启"允许通知"。
       - **Android**：进入"设置" > "应用" > "Google Media" > "通知"，确认通知权限已启用。

    2. **检查应用内通知设置**：登录账户，进入**账户** > **通知设置**，确认您已订阅相关通知类别。

    3. **检查勿扰模式**：确认设备未开启勿扰模式或夜间模式，以免屏蔽推送通知。

    如需进一步帮助，请访问[故障排除](/support/troubleshooting)页面。
  </Accordion>
</AccordionGroup>
